# Spring AI Series

A hands-on series of Spring Boot projects exploring the [Spring AI](https://docs.spring.io/spring-ai/reference/) framework. Each module covers a distinct set of capabilities, progressing from basic chat integration to more advanced patterns.

## Modules

| Module | Description |
|---|---|
| [`getting-started`](./getting-started) | Core `ChatClient` API — plain responses, system prompts, streaming, and structured entity output. |

## Stack

- Spring Boot 4
- Spring AI 2.0
- Ollama (local LLM runtime)
